# Baṣīra · بصيرة

An offline, trilingual (English · العربية · Türkçe) devotional companion for caring for one's eyesight — through transmitted duʿās, Qurʾānic recitations, ṣalawāt, and counted practices, woven into the rhythm of the daily prayers.

**Live:** https://talgatdilmurat.github.io/basira/

A single, self-contained HTML file — no build step, no dependencies, fully offline. Add it to your home screen to use it as an app. Every practice carries its source.

*A devotional companion, not medical advice. Any change in vision, pain, flashes or floaters → see an ophthalmologist promptly.*
